## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/study-mentor.git
```

2. Navigate to the project directory

```bash
cd study-mentor
```

3. Install dependencies using npm

```bash
npm install
```

4. Run

```bash
npm run dev
```
